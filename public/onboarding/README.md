# Onboarding backgrounds

Drop 4 full-bleed phone mockup screenshots here for the login walkthrough:

- `01.png` — Dashboard / hero value prop
- `02.png` — Calendar with upcoming renewals
- `03.png` — Gmail detection screen
- `04.png` — Savings insights

Recommended size: **1080×1920** (or 2x for retina), portrait PNG or JPG.
The images are displayed edge-to-edge with `background-size: cover`, and the
bottom half is covered by a frosted glass card. Focus the subject in the
upper ~60% of the image so it remains visible.

If a file is missing the slide shows a light grey background until you add
the asset — nothing else breaks.
